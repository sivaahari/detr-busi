\# DETR for Breast Ultrasound Nodule Detection (BUSI)



This project implements a lightweight Detection Transformer (DETR) for detecting breast nodules in ultrasound images using the BUSI dataset.



\## Key Features

\- Lightweight DETR (ResNet18 backbone)

\- Prior-guided learning (statistical geometric priors)

\- Edge-based structural enhancement

\- Clean, modular PyTorch implementation



\## Motivation

Ultrasound images present unique challenges:

\- Speckle noise

\- Irregular nodule shapes

\- Blurred boundaries



This project explores how transformer-based detection can be adapted to medical imaging with domain priors.



\## Tech Stack

\- PyTorch
